Basic Data Exploration
in SQL

Data exploration using COVID-19 dataset in SQL. Data ranges from 2020 to 2025.
In this project, I explored the dataset to find out the number of cases in each country & over time. Covid deaths by continent was also investigated where Europe had the highest toll and Oceania, the least.
Brunei, San Marino, and Austria were the top three countries with the highest number of cases per population.
Meanwhile, Peru, Bulgaria, and North Macedonia had the highest number of deaths per population.
In addition, I explored the dataset to find out the Top 10 countries with the highest vaccinated population i.e. total vaccinated people per population.
In the end, I created a preliminary dashboard in Tableau to visualize an overview of the data.
